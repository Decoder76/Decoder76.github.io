## 🔧 Installation and Setup

To get started with this repository locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Decoder76/Decoder76.github.io.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Decoder76.github.io
   ```
3. Open the project in your favorite editor or deploy it to a web server to see the animations in action.
